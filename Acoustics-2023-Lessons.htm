This spreadsheet contains the weighted score for the whole set of 5 in-class tests of previous year 2022, scaled to 30. At the end of the course, it wil be replaced with the version for 2023.

Who reached 18/30 or above can do the oral exam directly. Instead who did not reach 18/30 need to pass the written test the day of the oral exam.

In any case all students can try to improve the score obtained with the in-class tests by performing the written exam test.
The better of the two scores will be taken into account.
