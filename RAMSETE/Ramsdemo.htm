Ramsete DEMO

From here You can download two DEMO files:

RAMSDEMO.ZIP

RAMSSHOW.ZIP

The first contains the demo version of two Ramsete programs, namely Ramsete CAD and Ramsete Graph. To try out them, extract the contents of the ZIP file in a directory (possibly called \RAMSETE), and run the programs by double-clicking on them from File Manager or Explorer.

The second contains a slide-show of some screenshots taken while working with Ramsete, which includes all the modules. This show is dos-based, but requires a VESA graphic board, so it is not certain that it will work on Your machine. We are working for a Windows show, but it is not ready yet... To run the slide show, extract all the files contained in the ZIP file into a temporary directory, then run the SHOW.BAT program.
