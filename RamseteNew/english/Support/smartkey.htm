Smartkey Drivers Download

Following these instructions, You can download the latest drivers which enable the use of the Smartkey dongle in any Windows operating systems (Windows 95, Windows 98, Windows ME, Windows NT, Windows 2000, Windows XP).

INSTRUCTIONS

You should download the file named SMARTKEY.ZIP.

Unzip it maintaining the sub-directories stored inside the ZIP archive and then look at the README.TXT file contained in the archive for more information abut the installation procedure.

The most updated drivers for Smartkey dongles can anyway be found at the Eutron Smartkey Home Page
