Hardlock

Following these instructions, You can download the latest drivers which enable the use of the Hardlock dongle in any Windows operating systems (Windows 95, Windows 98, Windows ME, Windows NT, Windows 2000, Windows XP).

Instructions

You can choose between a self-installing driver file named

HLDRV32.EXE
and a comprehensive ZIP file containing all the drivers, for manual installing

Alternatively, if You like to have control to what happens and where, You can choose the manual installation. In this case, download the third file, unzip it maintaining the sub-directories stored inside the ZIP archive and then look at the README.TXT file contained in the archive for more information abut the installation procedure.

The most updated drivers for Hardlock dongles can anyway be found at the Hardlock Home Page
