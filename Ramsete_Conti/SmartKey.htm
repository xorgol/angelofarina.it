Smartkey

Download the Latest Smartkey Drivers (Ramsete 2.xx with white dongle)

Following these instructions, you can download the latest drivers which enable the use of the Smartkey dongle under 32 bit operating systems (Windows 95, Windows 98, Windows NT, Windows 2000, Windows XP).

Instructions:

Go to the directory containing the dongle drivers:

www.ramsete.com/Public/ramsete/drivers

Here You find these driver files:

Parall-NT2kXP.zip    Parallel Port dongle, Windows NT, 2k, XP
Parall-Win9x.zip      Parallel port dongle, Windows 95, 98, 98SE, ME
USB.zip                     USB dongle, Win ME, 2k, XP
Visual32-SDI.zip      GUI program for setup/maintanance of the drivers (not recommended)
skeydrv.dll                VisualBasic low-level interface to the drivers. It must be present either in the Ramsete installation directory or in C:\Windows\system

Unzip the proper file maintaining the sub-directories stored inside the ZIP archive and then look at the README.TXT file contained in the archive for more information abut the installation procedure.

The most updated drivers for Smartkey dongles can anyway be found at the Eutron Smartkey Home Page
