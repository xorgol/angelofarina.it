Ramsete View

This program allows to display the results of the simulation process.At first it’s possible to load the geometry file of the room, including sources and receiver position as shown in the next Ramsete View screen.  
     
This program allows to map many different acoustical parameters. A easy to use user interface allows to select many options. Output map of any simulations can be displayed immediately on the program window or can create thanks to embedded interface to Surfer (® Golden Software inc.), one of the widespread programs for maps interpolation and processing. A GRD file is produced.  
     
The next Ramsete View screens show 2-D and 3-D view mapping.  
     
This program also allows to display the energetic impulse response and the ray paths view  
     
For a complete listing of the available tools, look at the Help files
