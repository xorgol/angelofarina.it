Smartkey

Download the Latest Smartkey Drivers (Ramsete 2.xx with white dongle)

Following these instructions, you can download the latest drivers which enable the use of the Smartkey dongle under Windows (95, 98, ME, NT, 2000, XP, Vista).

Instructions:

Go to the directory containing the dongle drivers:

http://www.angelofarina.it/Public/Ramsete/Drivers/Smartkey_(white)/

Here You find these files:

Driver_Installation_SDI.zip General Driver for Windows NT, 2k, XP, Vista
skeydrv.dll VisualBasic low-level interface to the drivers. It must be present in C:\Windows\system32.

Unzip the ZIP archive and then look at the README.TXT file contained in it for more information abut the installation procedure.

Note: Usually installing the general driver is everything you need to do. However, if you do not have the Smartkey dongle installed, and consequently you do not want to install the General Driver above, then manually copy the skeydrv.dll file in C:\Windows\System32 .

The most recently updated drivers for Smartkey dongles can be found at the Eutron Smartkey Home Page
