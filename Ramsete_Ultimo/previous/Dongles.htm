RAMSETE : Room Acoustics modeling on PC

Dongles

Hardlock Drivers (black dongle, Ramsete 1xx updated from previous versions)

Although on most computers (running Windows 9x) there is no need for specific drivers, on some machines (running NT-4, Win2000, XP) it is necessary to load a small software piece for having the Hardlock hardware protection dongle (black) to be recognized.
If this is your case, you find those drivers here.

 

Smartkey Drivers (white dongle, Ramsete 2.xx & 1.xx newly purchased)

Although on most computers (running Windows 9x) there is no need for specific drivers, on some machines (running NT-4, Win2000, XP) it is necessary to load a small software piece for having the Smartkey hardware protection dongle (white) to be recognized. A special driver is required also for users of the new Smartkey USB dongle. If this is your case, you find those drivers here.

 

New USB memory stick (driverless, Ramsete 3.xx)

Starting with the release of Ramsete 3.0 (31/08/2016) there is no need anymore of special drivers, as the new hardware protection is based on a standard USB memory stick, containing a special key. Please contact Spectra for upgrading your old dongle to the new USB memory stick.
