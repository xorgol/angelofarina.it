RAMSETE : Room Acoustics modelling on PC

Ramsete Tracer

This is the computation engine of the package. You can select any combination of parameters and start the computation of as many batch tasks as you want.  
     

During the computation you can see what's going on and eventually abort the procedure

For a complete listing of the available tools, look at the Help files
